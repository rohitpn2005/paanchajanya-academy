PAANCHAJANYA ACADEMY — IMAGES
=============================

Drop your own photos in here. KEEP THE SAME FILENAME shown below and the
site picks them up automatically (just overwrite the placeholder .jpg).
Recommended: same width/height ratio as listed, JPG or WEBP, well lit.
Tip: name your file exactly as the existing one, e.g. save your hall photo
as images/yoga/the-main-hall.jpg.

Only the WORKSHOPS page images are dynamic (managed from your Google Drive
or Sheet), so there is no workshops folder here.
